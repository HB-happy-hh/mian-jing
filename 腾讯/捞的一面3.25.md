## 第一题：最长递增子序列和

### 题目描述

给定一个整数数组，找出所有**严格递增子序列**中元素之和最大的那个，返回该最大和。

> 子序列：不要求连续，但需保持原数组中的相对顺序。
> 严格递增：子序列中每个元素必须严格大于前一个元素。

**输入格式：**

```
第一行：整数 n，表示数组长度
第二行：n 个整数，空格分隔
```

**输出格式：**

```
一个整数，表示最长递增子序列的最大元素和
```

---

### 示例

**示例 1：**

```
输入：
5
1 3 2 4 5

输出：
13
```

**解释：**
所有严格递增子序列中，`[1, 3, 4, 5]` 的和最大为 1+3+4+5 = **13**

**示例 2：**

```
输入：
6
3 1 2 1 5 4

输出：
11
```

**解释：**
`[1, 2, 5]` = 8，`[1, 2, 4]` = 7，`[3, 5]` = 8，`[1, 5]` = 6，`[3, 4]` = 7
最优：`[1, 2, 5]` = **11** 或 `[3, 5]` = 8，答案为 **11**

---

### 解题思路

#### DP 定义

```
dp[i] = 以 arr[i] 结尾的严格递增子序列的最大元素和
```

#### 状态转移

```
dp[i] = max(dp[j] + arr[i])，其中 j < i 且 arr[j] < arr[i]
        若不存在合法前驱，则 dp[i] = arr[i]（子序列只含自身）

答案 = max(dp[0], dp[1], ..., dp[n-1])
```

#### 过程推导（示例 1：`[1, 3, 2, 4, 5]`）


| i   | arr[i] | 可接的前驱 j                                          | dp[i]  |
| --- | ------ | ------------------------------------------------ | ------ |
| 0   | 1      | 无                                                | **1**  |
| 1   | 3      | j=0: dp[0]+3=4                                   | **4**  |
| 2   | 2      | j=0: dp[0]+2=3                                   | **3**  |
| 3   | 4      | j=0: 1+4=5，j=1: 4+4=**8**，j=2: 3+4=7             | **8**  |
| 4   | 5      | j=0: 1+5=6，j=1: 4+5=9，j=2: 3+5=8，j=3: 8+5=**13** | **13** |


答案 = max(1, 4, 3, 8, 13) = **13**，对应子序列 `[1, 3, 4, 5]`

---

### 完整代码

#### 方法一：O(n²) DP（推荐，清晰易懂）

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) arr[i] = sc.nextInt();

        long[] dp = new long[n];
        long ans = Long.MIN_VALUE;

        for (int i = 0; i < n; i++) {
            dp[i] = arr[i]; // 至少包含自身
            for (int j = 0; j < i; j++) {
                if (arr[j] < arr[i]) {
                    dp[i] = Math.max(dp[i], dp[j] + arr[i]);
                }
            }
            ans = Math.max(ans, dp[i]);
        }

        System.out.println(ans);
    }
}
```

- **时间复杂度：** O(n²)
- **空间复杂度：** O(n)

---

#### 方法二：O(n log n) 树状数组（进阶优化）

将值域坐标压缩后，用**树状数组**维护前缀最大 dp 值：

- `query(rank - 1)`：查询所有值 < arr[i] 中的最大 dp
- `update(rank, dp[i])`：更新当前值的 dp

```java
import java.util.*;

public class Main {
    static long[] bit;
    static int size;

    // 查询前缀最大值 [1, i]
    static long query(int i) {
        long res = 0;
        for (; i > 0; i -= i & -i) res = Math.max(res, bit[i]);
        return res;
    }

    // 更新位置 i 的值
    static void update(int i, long val) {
        for (; i <= size; i += i & -i) bit[i] = Math.max(bit[i], val);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) arr[i] = sc.nextInt();

        // 坐标压缩：将原始值映射为 1, 2, 3... 的排名
        int[] sorted = arr.clone();
        Arrays.sort(sorted);
        sorted = Arrays.stream(sorted).distinct().toArray();
        size = sorted.length;
        bit = new long[size + 1];

        long ans = Long.MIN_VALUE;
        for (int x : arr) {
            int rank = Arrays.binarySearch(sorted, x) + 1; // 1-indexed 排名
            long prev = query(rank - 1);                   // 所有比 x 小的元素中最大的 dp
            long cur = (prev > 0 ? prev : 0) + x;         // 当前 dp 值
            ans = Math.max(ans, cur);
            update(rank, cur);
        }

        System.out.println(ans);
    }
}
```

- **时间复杂度：** O(n log n)
- **空间复杂度：** O(n)

---

### 两种方法对比


| 方法      | 时间复杂度      | 空间复杂度 | 适用场景              |
| ------- | ---------- | ----- | ----------------- |
| DP 双重循环 | O(n²)      | O(n)  | n ≤ 10⁴，面试首选，代码简洁 |
| 树状数组    | O(n log n) | O(n)  | n ≤ 10⁵，追求最优      |
