Backtracking · LeetCode 46

全排列

给定不含重复数字的数组，返回所有可能的全排列。
核心算法：回溯（Backtracking）—— 沿决策树深入，走不下去就原路退回。

核心三步

①

选择

从未使用的数中选一个，加入当前路径，并标记为已使用。

②

递归

路径长度等于数组长度时，记录结果；否则继续深入下一层。

③

撤销（回溯）

递归返回后，把刚才加入的数移出路径，标记恢复未使用。

Java 代码

public List<List<Integer>> permute(int[] nums) {
List<List<Integer>> result = new ArrayList<>();
boolean[] used = new boolean[nums.length];
backtrack(nums, used, new ArrayList<>(), result);
return result;
}
 
void backtrack(int[] nums, boolean[] used,
List<Integer> path, List<List<Integer>> result) {
if (path.size() == nums.length) { // ← 终止条件
result.add(new ArrayList<>(path)); // ← 收集结果（拷贝！）
return;
}
for (int i = 0; i < nums.length; i++) {
if (used[i]) continue; // ← 跳过已使用
used[i] = true; // ← ① 选择
path.add(nums[i]);
backtrack(nums, used, path, result); // ← ② 递归
used[i] = false; // ← ③ 撤销
path.remove(path.size() - 1);
}
}
