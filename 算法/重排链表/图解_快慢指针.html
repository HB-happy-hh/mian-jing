条件缺失会怎样？
写法 偶数 [1,2,3,4] 奇数 [1,2,3,4,5] 结论
两个条件都写 ✅ slow 停在 2 ✓ slow 停在 3 ✓ 正确
只写 fast.next != null slow 停在 2 ✓ slow 跑到 4，越过中点 ✗ 奇数时出错
只写 fast.next.next != null fast.next 为 null 时访问 .next → NullPointerException ✗ slow 停在 3 ✓ 偶数时崩溃
