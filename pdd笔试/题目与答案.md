## 第一题：卡车运输货物

### 题目描述

多多需要驾驶卡车按照指定的路线运输货物，线路沿途设有多个途径点，每个途径点会对卡车的载重产生影响，有些途径点需要装载货物（增加载重），有些途径点需要卸载货物（减少载重）。若卡车货物载重小于途径点需卸载货物重量，则车辆全量卸载后前往下一个途径点。卡车有初始载重 `initialWeight`，为了确保行车安全，卡车的载重必须始终小于等于最大载重 `maxSafeWeight` 进行运输。

需要你帮忙高效计算下最大的连续途径点个数，多多在这些连续途径点始终保持载重不超过安全限制。如果所有途径点都没有安全运输，则输出 `-1`。

**输入描述：**

共两行：
1. 第一行包含 3 个整数 `initialWeight`、`maxWeight`、`N`，分别表示卡车初始载重、卡车最大安全载重、途径点数量（1 ≤ maxWeight ≤ 100000，1 ≤ n ≤ 100000，0 ≤ initialWeight ≤ maxWeight）
2. 第二行包含 N 个整数 `weights[n]`，表示途径点对卡车实际载重的影响，正数为装载，负数为卸载，对于任意一个途径点均有 -100 ≤ weights[i] ≤ 100

**输出描述：**

共一行，最大的连续途径点个数，使得卡车在这些连续途径点始终保持载重不超过安全限制，如果所有途径点都没有安全运输，则输出 `-1`。

**示例：**

输入：
```
5 9 4
3 2 -4 3
```

输出：
```
2
```

说明：
- 途径点A：载重 = 5+3 = 8，未超过最大安全载重
- 途径点B：载重 = 8+2 = 10，超过最大安全载重
- 途径点C：载重 = 10-4 = 6，未超过最大安全载重
- 途径点D：载重 = 6+3 = 9，未超过最大安全载重
- 最大连续安全段为 C 和 D，返回 2

---

### 解题思路

1. 模拟每个途径点后的实际载重（若当前载重 < 需卸载量，则全量卸载归零）
2. 线性扫描，统计连续满足 `weight ≤ maxSafeWeight` 的最长子序列

**时间复杂度：** O(n)  
**空间复杂度：** O(n)

### 答案代码

```java
import java.util.Scanner;

public class Main {

    public static int solution(int initialWeight, int maxSafeWeight, int n, int[] weights) {

        // 用来记录每个途径点经过后的实际载重
        int[] actualWeight = new int[n];

        // 当前载重，一开始等于初始载重
        int cur = initialWeight;

        // 遍历每个途径点，模拟载重变化
        for (int i = 0; i < n; i++) {
            if (weights[i] < 0 && cur < -weights[i]) {
                // 需要卸的货比车上的货还多，直接全部卸完，载重变0
                cur = 0;
            } else {
                // 正常情况：装货就加，卸货就减
                cur = cur + weights[i];
            }
            // 记录这个途径点结束后的载重
            actualWeight[i] = cur;
        }

        // 用来记录最长连续安全段的长度
        int maxLen = 0;
        // 用来记录当前正在统计的连续安全段长度
        int curLen = 0;

        // 遍历每个途径点的实际载重，找最长连续安全段
        for (int i = 0; i < n; i++) {
            if (actualWeight[i] <= maxSafeWeight) {
                // 这个途径点安全，连续段加1
                curLen = curLen + 1;
                // 如果当前连续段比最大值还长，就更新最大值
                if (curLen > maxLen) {
                    maxLen = curLen;
                }
            } else {
                // 这个途径点不安全，连续段断掉，重新从0开始数
                curLen = 0;
            }
        }

        // 如果最大值还是0，说明没有一个安全的途径点，返回-1
        if (maxLen == 0) {
            return -1;
        } else {
            return maxLen;
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 读取第一行：初始载重、最大安全载重、途径点数量
        int initialWeight = sc.nextInt();
        int maxSafeWeight = sc.nextInt();
        int n = sc.nextInt();

        // 读取第二行：每个途径点对载重的影响
        int[] weights = new int[n];
        for (int i = 0; i < n; i++) {
            weights[i] = sc.nextInt();
        }

        // 调用 solution 方法并输出结果
        int result = solution(initialWeight, maxSafeWeight, n, weights);
        System.out.println(result);
    }
}
```

## 第二题：课程排课计划

### 题目描述

多多的学校有 n 门课程，编号从 1 到 n。每门课程可能有一门直接先修课程（或者没有）。如果至少满足以下条件之一，则一门课程 A 是另一门课程 B 的先修课程：

1. 课程 A 是课程 B 的直接先修课程
2. 课程 B 有一门直接先修课程 C，且课程 A 是课程 C 的先修课程

现在，多多需要安排这些课程的排课计划，将所有 n 门课程分配到若干个学期。每个学期中，不能有两门课程 A 和 B 使得 A 是 B 的先修课程。问最少需要多少个学期？

**输入描述：**

- 第一行包含整数 n（1 ≤ n ≤ 2000），课程数量
- 接下来 n 行，每行包含一个整数 pi（1 ≤ pi ≤ n 或 pi = -1），第 i 行的 pi 表示第 i 门课程的直接先修课程，如果 pi 为 -1，表示该课程没有先修课程

保证没有课程是自己的直接先修课程（pi ≠ i），也不存在循环依赖。

**输出描述：**

输出一个整数，表示最少需要的学期数。

**示例：**

输入：
```
5
-1
1
2
1
-1
```

输出：
```
3
```

说明：最少需要3个学期：
- 组1：课程1, 5
- 组2：课程2, 4
- 组3：课程3

---

### 解题思路

每门课程最多只有一个直接先修课，所以所有课程形成一片**森林（多棵树）**。

同一学期中不能有先修关系，也就是同一条链上的课程必须分属不同学期。

因此：**最少学期数 = 最长链的长度（树的最大深度）**

用递归+缓存（记忆化）计算每门课的深度，答案就是最大深度。

**时间复杂度：** O(n)  
**空间复杂度：** O(n)

### 答案代码

```java
import java.util.Scanner;

public class Main {

    // 存储每门课的直接先修课（-1 表示没有）
    static int[] parent;
    // 存储每门课的深度（0 表示还没算过）
    static int[] depth;

    // 递归计算课程 i 的深度，用 depth[] 缓存避免重复计算
    public static int computeDepth(int i) {
        // 如果已经算过了，直接返回缓存结果
        if (depth[i] != 0) {
            return depth[i];
        }
        // 没有先修课，深度为1（第1学期就能上）
        if (parent[i] == -1) {
            depth[i] = 1;
        } else {
            // 有先修课，深度 = 先修课的深度 + 1
            depth[i] = computeDepth(parent[i]) + 1;
        }
        return depth[i];
    }

    public static int solution(int n) {
        // 计算所有课程的深度
        for (int i = 1; i <= n; i++) {
            computeDepth(i);
        }

        // 找最大深度，即为最少学期数
        int maxDepth = 0;
        for (int i = 1; i <= n; i++) {
            if (depth[i] > maxDepth) {
                maxDepth = depth[i];
            }
        }
        return maxDepth;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 读取课程数量
        int n = sc.nextInt();

        // parent[i] 表示课程 i 的直接先修课编号，-1 表示没有
        parent = new int[n + 1];
        depth = new int[n + 1]; // 初始全为0，表示未计算

        // 读取每门课程的先修课
        for (int i = 1; i <= n; i++) {
            parent[i] = sc.nextInt();
        }

        System.out.println(solution(n));
    }
}
```

## 第三题：背包装奖品

### 题目描述

辰辰参加某知名的在线答题栏目获得了冠军，节目组提供了 N 种奖品，每种奖品的价值记为 Vi；同时节目组提供给辰辰两个背包，每种奖品最多只能放入其中的一个背包里；另外，刁钻的节目组还给辰辰设置了一个额外的障碍，就是每个背包中任意两件奖品的价值差不能超过 T。

在这样的约束下，辰辰最多能带走多少件奖品？

**输入描述：**

- 第一行输入两个正整数 N 和 T，其中 N 代表节目组提供奖品的数量，T 代表约定的同一个背包里任意两件奖品价值差的阈值
- 接下来输入 N 行，每行一个数 Vi 代表相应的奖品价值
- （N ≤ 50000，T ≤ 50000000，Vi ≤ 50000000）

**输出描述：**

输出一个正整数，代表辰辰最多能带走的奖品件数。

**示例：**

输入：
```
6 3
5
4
2
1
8
10
```

输出：
```
5
```

---

### 解题思路

**关键观察：**
- 排序后，同一个背包里的奖品价值差最大的是最大值减最小值，所以同一背包的奖品必然是排序后**连续的一段**
- 两个背包 = 把排序后的数组分成**两段不重叠的连续区间**，每段内最大值-最小值 ≤ T

**算法步骤：**
1. 排序
2. 双指针求 `left[i]`：以 i 结尾的最长合法段长度
3. 双指针求 `right[i]`：以 i 开头的最长合法段长度
4. 枚举分割点，答案 = `max(prefixLeft[i] + suffixRight[i+1])`

排序后 `[1,2,4,5,8,10]`，T=3，在分割点 i=2 时：左背包 {1,2,4}=3个，右背包 {5,8}=2个，合计 **5** ✓

**时间复杂度：** O(n log n)（主要是排序）  
**空间复杂度：** O(n)

### 答案代码

```java
import java.util.Arrays;
import java.util.Scanner;

public class Main {

    public static int solution(int n, long t, long[] v) {
        // 第一步：排序，排序后同一背包内的奖品必然是连续的一段
        Arrays.sort(v);

        // 第二步：双指针计算 left[i]：以第i个奖品结尾的合法段最大长度
        int[] left = new int[n];
        int l = 0;
        for (int r = 0; r < n; r++) {
            // 如果窗口内价值差超过t，左边界右移
            while (v[r] - v[l] > t) {
                l = l + 1;
            }
            left[r] = r - l + 1;
        }

        // 第三步：双指针计算 right[i]：以第i个奖品开头的合法段最大长度
        int[] right = new int[n];
        int r = 0;
        for (l = 0; l < n; l++) {
            // r至少要和l一样大
            if (r < l) {
                r = l;
            }
            // 右边界尽量往右扩
            while (r + 1 < n && v[r + 1] - v[l] <= t) {
                r = r + 1;
            }
            right[l] = r - l + 1;
        }

        // 第四步：计算前缀最大值 prefixLeft[i] = left[0..i] 中的最大值
        int[] prefixLeft = new int[n];
        prefixLeft[0] = left[0];
        for (int i = 1; i < n; i++) {
            if (left[i] > prefixLeft[i - 1]) {
                prefixLeft[i] = left[i];
            } else {
                prefixLeft[i] = prefixLeft[i - 1];
            }
        }

        // 第五步：计算后缀最大值 suffixRight[i] = right[i..n-1] 中的最大值
        int[] suffixRight = new int[n];
        suffixRight[n - 1] = right[n - 1];
        for (int i = n - 2; i >= 0; i--) {
            if (right[i] > suffixRight[i + 1]) {
                suffixRight[i] = right[i];
            } else {
                suffixRight[i] = suffixRight[i + 1];
            }
        }

        // 第六步：枚举分割点，找两个背包能带走的最多奖品数
        // 先考虑只用一个背包的情况
        int ans = prefixLeft[n - 1];

        // 再枚举分割点：左背包用 [0..i]，右背包用 [i+1..n-1]
        for (int i = 0; i < n - 1; i++) {
            int total = prefixLeft[i] + suffixRight[i + 1];
            if (total > ans) {
                ans = total;
            }
        }

        return ans;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        long t = sc.nextLong();

        long[] v = new long[n];
        for (int i = 0; i < n; i++) {
            v[i] = sc.nextLong();
        }

        System.out.println(solution(n, t, v));
    }
}
```
