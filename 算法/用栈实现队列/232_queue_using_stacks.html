💡 核心思路
  • push(x) 直接压入 stack1（入队栈），始终 O(1)
  • pop() 若 stack2 非空，直接弹出 stack2 栈顶；
    若 stack2 为空，先把 stack1 全部倒入 stack2，再弹出
  • peek() 逻辑同 pop，只看不取，返回 stack2 栈顶值
  • empty() stack1 && stack2 均为空时返回 true
  • 倒入时把 stack1 的逆序变成 stack2 的正序， 恰好满足 FIFO ✓ 每个元素最多被搬运一次 → 均摊 O(1)

// LeetCode 232 · 用栈实现队列
import java.util.Stack;

class MyQueue {

    private Stack<Integer> stack1;  // 入队栈：push 专用
    private Stack<Integer> stack2;  // 出队栈：pop / peek 专用

    public MyQueue() {
        stack1 = new Stack<>();
        stack2 = new Stack<>();
    }

    /** 将元素 x 推到队列末尾，O(1) */
    public void push(int x) {
        stack1.push(x);
    }

    /** 从队列开头移除并返回元素，均摊 O(1) */
    public int pop() {
        transfer();       // 懒惰转移：s2 为空才搬
        return stack2.pop();
    }

    /** 返回队列开头的元素（不移除），均摊 O(1) */
    public int peek() {
        transfer();
        return stack2.peek();
    }

    /** 队列是否为空，O(1) */
    public boolean empty() {
        return stack1.isEmpty() && stack2.isEmpty();
    }

    /** 仅当 stack2 为空时，将 stack1 全部倒入 stack2 */
    private void transfer() {
        if (stack2.isEmpty()) {
            while (!stack1.isEmpty()) {
                stack2.push(stack1.pop());
            }
        }
    }
}

/*
 * 调用示例：
 *   MyQueue q = new MyQueue();
 *   q.push(1);  q.push(2);  q.push(3);
 *   q.peek()  → 1      （stack2 此时 [3,2,1]↑，栈顶=1）
 *   q.pop()   → 1
 *   q.pop()   → 2
 *   q.empty() → false
 */
⏱ 时间 & 空间复杂度
操作最坏情况均摊复杂度说明
push(x) O(1) O(1) 直接压入 stack1
pop() O(n) O(1) ★ stack2 为空时触发一次性转移，但每元素仅被搬运一次
peek() O(n) O(1) ★ 同 pop，只看不取
empty() O(1) O(1) 两栈均判空
空间复杂度 O(n)，两栈合计存储 n 个元素
