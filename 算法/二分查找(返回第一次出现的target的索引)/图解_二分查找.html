left > right，循环结束。返回 result = 1，即 target=2 第一次出现的位置。
核心思路 与普通二分唯一的区别
普通二分：找到 target 就 立即 return mid
本题：找到 target 后，记录 result = mid，然后 right = mid − 1 强制向左收缩
直到区间耗尽，result 就是最左边（第一次）出现的位置
