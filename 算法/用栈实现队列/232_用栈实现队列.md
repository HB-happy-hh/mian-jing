# 232. 用栈实现队列

**难度**：简单  
**标签**：栈、队列、设计

---

## 题目

请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（`push`、`pop`、`peek`、`empty`）：

- `void push(int x)` 将元素 x 推到队列的末尾
- `int pop()` 从队列的开头移除并返回元素
- `int peek()` 返回队列开头的元素
- `boolean empty()` 如果队列为空，返回 `true`；否则，返回 `false`

---

## 解题思路：双栈 · 懒惰转移法

### 核心思想

用两个栈角色分工：

| 角色 | 栈名 | 职责 |
|------|------|------|
| 入队栈 | `stack1` | 只负责 `push` |
| 出队栈 | `stack2` | 只负责 `pop` / `peek` |

### 图解过程

**操作序列：push(1) → push(2) → push(3) → pop() → push(4) → pop()**

```
【push(1) push(2) push(3)】
  stack1（入队）       stack2（出队）
  ┌───────┐            ┌───────┐
  │   3   │  ← TOP     │       │
  │   2   │            │       │
  │   1   │            │       │
  └───────┘            └───────┘
  队头 = 1（最底部）

【pop() 时：stack2 为空 → 把 stack1 全部倒入 stack2】
  stack1（入队）       stack2（出队）
  ┌───────┐            ┌───────┐
  │       │            │   1   │  ← TOP = 队头 ✓
  │       │   ⟹⟹⟹    │   2   │
  │       │            │   3   │
  └───────┘            └───────┘
  弹出 stack2 栈顶 = 1  ← 正是队头！

【push(4)：继续压入 stack1，stack2 原有元素不受影响】
  stack1（入队）       stack2（出队）
  ┌───────┐            ┌───────┐
  │   4   │  ← TOP     │   2   │  ← TOP
  │       │            │   3   │
  └───────┘            └───────┘

【pop()：stack2 非空，直接弹出 → 2】
```

**关键：** 倒入时 stack1 的**逆序**恰好变成 stack2 的**正序**，完美还原 FIFO 顺序。

## Java 代码

```java
import java.util.Stack;

class MyQueue {
    private Stack<Integer> stack1;  // 入队栈
    private Stack<Integer> stack2;  // 出队栈

    public MyQueue() {
        stack1 = new Stack<>();
        stack2 = new Stack<>();
    }

    // push：直接压入 stack1，O(1)
    public void push(int x) {
        stack1.push(x);
    }

    // pop：均摊 O(1)
    public int pop() {
        transfer();
        return stack2.pop();
    }

    // peek：均摊 O(1)
    public int peek() {
        transfer();
        return stack2.peek();
    }

    // empty：O(1)
    public boolean empty() {
        return stack1.isEmpty() && stack2.isEmpty();
    }

    // 仅当 stack2 为空时，才把 stack1 全部倒入
    private void transfer() {
        if (stack2.isEmpty()) {
            while (!stack1.isEmpty()) {
                stack2.push(stack1.pop());
            }
        }
    }
}
```

---

## 复杂度分析

| 操作 | 最坏 | **均摊** | 说明 |
|------|------|----------|------|
| `push` | O(1) | **O(1)** | 直接压栈 |
| `pop` | O(n) | **O(1)** ★ | 转移仅发生一次，每个元素最多被搬一次 |
| `peek` | O(n) | **O(1)** ★ | 同 pop |
| `empty` | O(1) | **O(1)** | 双栈判空 |
| 空间 | — | **O(n)** | 两栈合计存 n 个元素 |

### 均摊 O(1) 的证明思路

每个元素的生命周期只有两次操作：
1. 从 `push` 进入 **stack1**（1次压栈）
2. 从 stack1 转移到 **stack2**（1次弹出 + 1次压入）

所以 n 次操作总共最多 3n 次栈操作 → 均摊每次 **O(1)**。
