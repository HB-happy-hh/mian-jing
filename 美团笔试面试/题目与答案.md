## 一、操作系统

### Q1：下列哪个数据结构不属于设备分配（单选）

- A 系统设备表
- B 控制器控制表
- C 逻辑控制表
- D 设备控制表

**答案：C**

**解析：**
设备分配使用的数据结构包括：
- **系统设备表（SDT）**：记录系统中所有设备信息
- **设备控制表（DCT）**：记录每台设备的状态及等待队列
- **控制器控制表（COCT）**：记录控制器信息
- **通道控制表（CHCT）**：记录通道信息

**逻辑控制表**并不是设备分配中的数据结构，属于干扰项。

## 三、数据结构

### Q3：在一棵度为4的树T中，若有10个度为4的结点，15个度为3的结点，4个度为2的结点，9个度为1的结点，则树T的叶结点个数是（单选）

- A 93
- B 65
- C 73
- D 64

**答案：B，65**

**解析：**

树中所有结点的**度之和 = 总结点数 - 1**（每条边对应一个孩子）

设叶结点（度为0）有 $n_0$ 个，总结点数：

$$n = 10 + 15 + 4 + 9 + n_0 = 38 + n_0$$

所有结点度之和：

$$10 \times 4 + 15 \times 3 + 4 \times 2 + 9 \times 1 = 40 + 45 + 8 + 9 = 102$$

由 **度之和 = 总结点数 - 1**：

$$102 = (38 + n_0) - 1 \Rightarrow n_0 = 65$$

### Q4：在升序有序表中进行折半查找，能构成折半查找中关键字比较序列的是（单选）

- A 50, 65, 57, 47, 54
- B 50, 65, 47, 62, 63
- C 50, 98, 73, 79, 84
- D 50, 20, 6, 21, 18

**答案：C**

**解析：**

折半查找每次比较后，下一个比较值必须在当前确定的范围内（维护搜索区间 `[lo, hi]`）：

| 选项 | 验证过程 | 结论 |
|------|---------|------|
| A | 50→右，65→左→(50,65)，57→左→(50,57)，**47<50越界** | ❌ |
| B | 50→右，65→左→(50,65)，**47<50越界** | ❌ |
| C | 50→右，98→左→(50,98)，73→右→(73,98)，79→右→(79,98)，84∈(79,98) | ✅ |
| D | 50→左，20→右→(20,50)，**6<20越界** | ❌ |

## 五、算法题（ACM模式）

### Q6：最小化序列和

**题目：**
给定长度为 $n$ 的整数序列。每个位置最多做一次操作1（$a_i \to \lfloor a_i/2 \rfloor$），最多做一次操作2（$a_i \to a_i - k$）。操作1共可用 $a$ 次，操作2共可用 $b$ 次。使最终序列所有元素之和尽可能小。

**思路：**
- 操作1对元素 $a_i$ 的收益固定为 $\lceil a_i/2 \rceil$，与是否做操作2无关
- 操作2每次固定减少 $k$，对任意元素效果相同
- 两种操作收益相互独立，贪心：操作1选收益最大的 $a$ 个元素；操作2用满 $b$ 次

**答案（Java）：**

```java
import java.util.*;
import java.io.*;

public class Main {

    static long solution(int n, int a, int b, long k, long[] arr) {
        long sum = 0;
        for (int i = 0; i < n; i++) sum += arr[i];

        long[] gain = new long[n];
        for (int i = 0; i < n; i++) gain[i] = (arr[i] + 1) / 2;

        Arrays.sort(gain);

        long op1 = 0;
        for (int i = n - 1; i >= n - a; i--) op1 += gain[i];

        long op2 = (long) b * k;
        return sum - op1 - op2;
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        int T = Integer.parseInt(br.readLine().trim());
        for (int i = 0; i < T; i++) {
            StringTokenizer st = new StringTokenizer(br.readLine());
            int n = Integer.parseInt(st.nextToken());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());
            long k = Long.parseLong(st.nextToken());
            long[] arr = new long[n];
            st = new StringTokenizer(br.readLine());
            for (int j = 0; j < n; j++) arr[j] = Long.parseLong(st.nextToken());
            sb.append(solution(n, a, b, k, arr)).append('\n');
        }
        System.out.print(sb);
    }
}
```

### Q7：最大交替和子序列

**题目：**
给定长度为 $n$ 的整数数组，从中选出一个子序列 $b_1, b_2, \ldots, b_m$，使得目标值 $F(b) = \sum_{i=1}^{m}(-1)^{i-1}b_i = b_1 - b_2 + b_3 - \cdots$ 最大。

**思路（DP）：**

维护两个状态：
- `bestPos`：当前最优，最后一个元素在**奇数位（+）**
- `bestNeg`：当前最优，最后一个元素在**偶数位（-）**

转移：
- `a[i]` 放 + 位：`dpPos = a[i] + max(0, bestNeg)`（前面可以没有元素）
- `a[i]` 放 - 位：`dpNeg = -a[i] + bestPos`（前面**必须**有 + 位元素）

答案 = `max(0, bestPos, bestNeg)`

**答案（Java）：**

```java
import java.util.*;
import java.io.*;

public class Main {

    static long solution(int n, long[] arr) {
        long bestPos = Long.MIN_VALUE / 2;
        long bestNeg = Long.MIN_VALUE / 2;
        long ans = 0;
        for (int i = 0; i < n; i++) {
            long dpPos = arr[i] + Math.max(0, bestNeg);
            long dpNeg = -arr[i] + bestPos;
            bestPos = Math.max(bestPos, dpPos);
            bestNeg = Math.max(bestNeg, dpNeg);
            ans = Math.max(ans, Math.max(dpPos, dpNeg));
        }
        return ans;
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        int T = Integer.parseInt(br.readLine().trim());
        for (int i = 0; i < T; i++) {
            int n = Integer.parseInt(br.readLine().trim());
            StringTokenizer st = new StringTokenizer(br.readLine());
            long[] arr = new long[n];
            for (int j = 0; j < n; j++) arr[j] = Long.parseLong(st.nextToken());
            sb.append(solution(n, arr)).append('\n');
        }
        System.out.print(sb);
    }
}
```

**验证：**
| 输入 | 输出 |
|------|------|
| [1,2,3] | 3（取3） |
| [-1,2,-3,4,-5] | 14（取2,-3,4,-5 → 2+3+4+5=14） |
| [5,5,5,5] | 5 |

### Q8：区间端点和大于区间最大值的区间数

**题目：**
给定长度为 $n$ 的序列 $v_1, v_2, \ldots, v_n$（$1 \le v_i \le 10^6$），统计满足 $v_i + v_j > \max_{k=i}^{j} v_k$ 的区间 $[i,j]$（$1 \le i < j \le n$）的数量。

**思路（分治 + 树状数组，O(n log²n)）：**

对跨中点的区间 $[i,j]$（$i \le mid < j$），最大值为 $\max(lmax[i], rmax[j])$，分两种情形：

- **情形A**：$lmax[i] \ge rmax[j]$，条件变为 $v[j] > lmax[i] - v[i]$
- **情形B**：$rmax[j] > lmax[i]$，条件变为 $v[i] > rmax[j] - v[j]$

由于 $lmax$、$rmax$ 单调，用双指针 + 树状数组在 $O(n \log n)$ 每层内统计。

**答案（Java）：**

```java
import java.util.*;
import java.io.*;

public class Main {
    static final int MAX_VAL = 1_000_000;
    static int[] bit = new int[MAX_VAL + 2];
    static int bitTotal = 0;
    static int[] v;
    static long ans = 0;

    static void update(int x, int val) {
        bitTotal += val;
        for (; x <= MAX_VAL; x += x & -x) bit[x] += val;
    }

    static int query(int x) {
        int s = 0;
        for (; x > 0; x -= x & -x) s += bit[x];
        return s;
    }

    static int countGreater(int threshold) {
        if (threshold <= 0) return bitTotal;
        if (threshold >= MAX_VAL) return 0;
        return bitTotal - query(threshold);
    }

    static void solve(int l, int r) {
        if (l >= r) return;
        int mid = (l + r) >>> 1;
        solve(l, mid);
        solve(mid + 1, r);

        int leftLen = mid - l + 1, rightLen = r - mid;
        int[] lmax = new int[leftLen];
        lmax[0] = v[mid];
        for (int k = 1; k < leftLen; k++) lmax[k] = Math.max(lmax[k - 1], v[mid - k]);
        int[] rmax = new int[rightLen];
        rmax[0] = v[mid + 1];
        for (int k = 1; k < rightLen; k++) rmax[k] = Math.max(rmax[k - 1], v[mid + 1 + k]);

        // 情形A：lmax[io] >= rmax[jo]，统计 v[j] > lmax[io]-v[i]
        int[] addedJ = new int[rightLen];
        int addedJCount = 0, pj = -1;
        for (int io = 0; io < leftLen; io++) {
            int li = lmax[io];
            while (pj + 1 < rightLen && rmax[pj + 1] <= li) {
                pj++;
                addedJ[addedJCount++] = v[mid + 1 + pj];
                update(v[mid + 1 + pj], 1);
            }
            ans += countGreater(li - v[mid - io]);
        }
        for (int k = 0; k < addedJCount; k++) update(addedJ[k], -1);

        // 情形B：rmax[jo] > lmax[io]，统计 v[i] > rmax[jo]-v[j]
        int[] addedI = new int[leftLen];
        int addedICount = 0, pi = -1;
        for (int jo = 0; jo < rightLen; jo++) {
            int rj = rmax[jo];
            while (pi + 1 < leftLen && lmax[pi + 1] < rj) {
                pi++;
                addedI[addedICount++] = v[mid - pi];
                update(v[mid - pi], 1);
            }
            ans += countGreater(rj - v[mid + 1 + jo]);
        }
        for (int k = 0; k < addedICount; k++) update(addedI[k], -1);
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine().trim());
        v = new int[n];
        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int i = 0; i < n; i++) v[i] = Integer.parseInt(st.nextToken());
        solve(0, n - 1);
        System.out.println(ans);
    }
}
```
