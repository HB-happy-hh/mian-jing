## 第一题：IOI 排名查询

### 题目描述

你正在参加一场 IOI 赛制的比赛，每个题目可以多次提交，取得分最高的一次计分，总分即为所有题目最高分之和。现在，有若干道独立的题目，按时间顺序依次有 n 次提交，第 i 次提交记为三元组 (a_i, b_i, c_i)，表示用户 a_i 在题目 b_i 上获得分数 c_i。

在每次提交处理完后，需要报告你（记编号为 1）的名次。排名采用如下规则：
- 若总分不同，则总分高者排名在前
- 若总分相同，则并列相同名次，但占用多个名次位置（例如，分数前四高的人的分数分别为 100, 100, 100, 80，则他们的排名为 1, 1, 1, 4）

**输入描述：**

- 第一行输入一个整数 n（1 ≤ n ≤ 2×10^5），表示提交记录的数量
- 此后 n 行，第 i 行输入三个整数 a_i, b_i, c_i（1 ≤ a_i ≤ 100；1 ≤ b_i ≤ 100；0 ≤ c_i ≤ 100），表示第 i 次提交记录

**输出描述：**

对于每一次提交，输出一个整数，表示第 i 次记录后你的排名。

**示例：**

输入：
```
10
2 1 0
1 1 80
3 2 100
1 2 60
3 1 40
3 3 60
1 1 90
5 1 100
5 2 100
1 4 50
```

输出：
```
1
1
2
1
1
2
2
2
3
1
```

---

### 解题思路

- 每个用户每道题只取历史最高分，总分 = 所有题最高分之和
- 每次提交后计算用户1的排名 = 比用户1总分高的人数 + 1
- 用户最多100人，每次更新后暴力遍历100人求排名，O(100) 完全够

**时间复杂度：** O(n × 100)  
**空间复杂度：** O(100 × 100)

> 注意：`best` 和 `current` 必须是**类的静态变量**，不能放在方法里，否则每次调用都会重置为0，丢失历史数据。

### 答案代码

```java
import java.util.Scanner;

public class Main {

    // 静态变量：整个程序运行期间只创建一次，能一直保留历史数据
    // best[用户][题目] = 该用户在该题上的历史最高分
    static int[][] best = new int[101][101];
    // total[用户] = 该用户的当前总分
    static int[] total = new int[101];

    // 处理一次提交，返回用户1当前的排名
    public static int solution(int a, int b, int c) {
        // 如果本次得分高于该用户在该题的历史最高分，才更新
        if (c > best[a][b]) {
            // 总分 = 原总分 - 旧最高分 + 新最高分
            total[a] = total[a] - best[a][b] + c;
            // 更新历史最高分
            best[a][b] = c;
        }

        // 计算用户1的排名：有多少人的总分严格大于用户1，就排在用户1前面
        int rank = 1;
        for (int i = 1; i <= 100; i++) {
            if (total[i] > total[1]) {
                rank = rank + 1;
            }
        }
        return rank;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 读取提交次数
        int n = sc.nextInt();

        for (int i = 0; i < n; i++) {
            int a = sc.nextInt(); // 用户编号
            int b = sc.nextInt(); // 题目编号
            int c = sc.nextInt(); // 本次得分

            // 每次提交后立刻输出当前排名
            System.out.println(solution(a, b, c));
        }
    }
}
```

## 第二题：字符串字典序最小重排

### 题目描述

游游有一个长度为 n 的字符串 s（下标从 1 开始，且仅由小写字母构成），以及一个长度为 n 的数组 {a_1, a_2, ..., a_n}。

游游希望你对数组进行重排。随后，游游将使用重排后的数组（记重排后的数组为 a'），按照 1 ~ n 的顺序，从空串开始拼接生成一个新的字符串 t：
- 第 i 次操作在 t 的末尾拼接 a'_i 个 s_i

游游希望最终得到的字符串 t 的字典序尽可能小。请你输出使得 t 的字典序最小化时，重排后的数组，如果有多个答案你可以输出任意一种。

**【名词解释】**

- **重排**：将数组中的元素以任意顺序重新排列，得到与原数组长度相同、元素集合相同的新数组
- **字符串的字典序比较**：从左到右逐个比较两个字符串的字符。如果在某个位置上字符不同，比较这两个字符的 ASCII 码顺序，ASCII 码小的字符串字典序也小。如果一直比较到其中一个字符串结束，则较短的字符串字典序更小

**输入描述：**

每个测试文件均包含多组测试数据。第一行输入一个整数 T（1 ≤ T ≤ 10^4）代表数据组数，每组测试数据描述如下：
- 第一行输入一个整数 n（1 ≤ n ≤ 2×10^5），表示字符串与数组的长度
- 第二行输入一个长度为 n、仅由小写字母构成的字符串 s
- 第三行输入 n 个整数 a_1, a_2, ..., a_n（1 ≤ a_i ≤ 10^9），表示数组元素

除此之外，保证单个测试文件的 n 之和不超过 2×10^5。

**输出描述：**

对于每一组测试数据，新起一行，输出 n 个整数，表示使字符串 t 字典序最小化时重排后的数组。

**示例：**

输入：
```
2
5
aabcd
1 2 3 4 5
3
bac
1 1 2
```

输出：
```
4 5 3 2 1
1 2 1
```

说明：
- 第一个样例拼接得到的字符串 t 为 "aaaaaaaaaabbbccd"
- 第二个样例中拼接得到字符串 t 为 "baac"，可以证明不存在更好的拼接方式

---

### 解题思路

**核心规律**：字符越小（如 'a'）的位置，应该分配**越大**的计数，让字符串在小字符上多停留；字符越大（如 'z'）的位置，应该分配**越小**的计数，快速跳过。

**算法步骤：**
1. 将计数数组 a 从大到小排序
2. 将所有位置按字符升序分组（'a' 的位置放一组，'b' 的位置放一组，...）
3. 从 'a' 开始，依次把最大的计数分给字符最小的位置

**验证示例2**：s="bac", a=[1,1,2]，计数降序=[2,1,1]
- 'a' 的位置：pos=1 → 分配2（最大计数）
- 'b' 的位置：pos=0 → 分配1
- 'c' 的位置：pos=2 → 分配1
- 结果：a'=[1,2,1]，t="b"+"aa"+"c"="baac" ✓

**时间复杂度：** O(n log n)（排序）  
**空间复杂度：** O(n)

### 答案代码

```java
import java.util.Arrays;
import java.util.Scanner;

public class Main {

    public static int[] solution(int n, String s, int[] a) {
        // 第一步：复制数组 a，然后升序排序，再手动翻转成降序
        // 目的：字符最小的位置要分配最大的计数
        int[] sortedA = new int[n];
        for (int i = 0; i < n; i++) {
            sortedA[i] = a[i];
        }
        Arrays.sort(sortedA); // 升序
        // 手动翻转为降序
        for (int i = 0; i < n / 2; i++) {
            int tmp = sortedA[i];
            sortedA[i] = sortedA[n - 1 - i];
            sortedA[n - 1 - i] = tmp;
        }

        // 第二步：按字符分组，记录每个字符对应的原始位置
        // posCount[c] = 字符 c 对应的位置有多少个
        // pos[c][k]   = 字符 c 的第 k 个位置的原始下标
        int[] posCount = new int[26];
        for (int i = 0; i < n; i++) {
            int c = s.charAt(i) - 'a'; // 'a'->0, 'b'->1, ..., 'z'->25
            posCount[c] = posCount[c] + 1;
        }
        int[][] pos = new int[26][];
        for (int c = 0; c < 26; c++) {
            pos[c] = new int[posCount[c]];
        }
        int[] fillIdx = new int[26]; // 记录每组当前填到第几个了
        for (int i = 0; i < n; i++) {
            int c = s.charAt(i) - 'a';
            pos[c][fillIdx[c]] = i;
            fillIdx[c] = fillIdx[c] + 1;
        }

        // 第三步：从字符'a'开始，依次把最大的计数分给字符最小的位置
        int[] result = new int[n];
        int idx = 0; // sortedA 的指针，从最大计数开始取
        for (int c = 0; c < 26; c++) {
            for (int k = 0; k < posCount[c]; k++) {
                int originalPos = pos[c][k]; // 原始位置下标
                result[originalPos] = sortedA[idx];
                idx = idx + 1;
            }
        }

        return result;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 读取测试数据组数
        int T = sc.nextInt();

        for (int t = 0; t < T; t++) {
            int n = sc.nextInt();
            String s = sc.next();
            int[] a = new int[n];
            for (int i = 0; i < n; i++) {
                a[i] = sc.nextInt();
            }

            int[] result = solution(n, s, a);

            // 输出结果，数字之间用空格分隔
            for (int i = 0; i < n; i++) {
                if (i > 0) {
                    System.out.print(" ");
                }
                System.out.print(result[i]);
            }
            System.out.println();
        }
    }
}
```
