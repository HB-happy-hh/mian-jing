杨辉三角

每个数等于它上方左右两数之和；首尾永远是 1。

核心规律

首尾规律

每行第一个和最后一个元素固定为 1，不需要计算。

j == 0 || j == i → 1

中间规律

中间的每个数 = 上一行同位置 + 上一行前一位置。

prev[j-1] + prev[j]

public List<List<Integer>> generate(int numRows) {
List<List<Integer>> result = new ArrayList<>();
for (int i = 0; i < numRows; i++) { // 逐行构建
List<Integer> row = new ArrayList<>();
for (int j = 0; j <= i; j++) { // 逐列填数
if (j == 0 || j == i) {
row.add(1); // 首尾填 1
} else {
List<Integer> prev = result.get(i - 1);
row.add(prev.get(j-1) + prev.get(j)); // 左上 + 右上
}
}
result.add(row); // 完成一行，加入结果
}
return result;
}
