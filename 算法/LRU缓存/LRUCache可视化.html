LeetCode 146

LRU 缓存
哈希表 + 双向链表

LRU（Least Recently Used，最近最少使用）缓存要求 get 和 put 均在 O(1) 时间内完成。 核心思路：用哈希表实现 O(1) 查找，用双向链表维护访问顺序，两者合作完成所有操作。

01 — 核心思路
🗂️

HashMap  →  O(1) 查找

key → Node 直接定位节点地址。
没有哈希表，链表查找是 O(n)，无法满足题目要求。

⛓️

双向链表 → O(1) 移动 / 删除

节点持有 prev 和 next 指针，
可在 O(1) 完成摘除和插入到头部，无需遍历。

🔗

虚拟头尾节点 → 消除边界判断

链表两端各有一个哨兵节点 head（dummy）和 tail（dummy），
永远不为空，省去 null 判断，代码更简洁。

📌

顺序约定 → 头=最近，尾=最久

每次访问/写入，把对应节点移到 head.next；
容量满时，淘汰 tail.prev（最久未使用）。

04 — 算法步骤
G

get(key)

① 查 HashMap：未命中 → 返回 -1。
② 命中 → moveToHead(node)（先摘后插头）。
③ 返回 node.val。

P

put(key, value)

① key 已存在 → 更新 val，moveToHead。
② key 不存在 → 创建新节点，addToHead，map.put。
③ 若 size > capacity → 删 tail.prev，map.remove。

+

addToHead(node)

将节点插到 head.next 位置（最近端）：
node.prev = head
node.next = head.next
head.next.prev = node
head.next = node

−

removeNode(node)

从链表中摘除节点（不销毁），绕过它：
node.prev.next = node.next
node.next.prev = node.prev
双向指针使得无需从头遍历，直接 O(1)。

05 — 核心代码（Java）
  LRUCache.java 
class LRUCache {
    private static class Node {
        int key, val;
        Node prev, next;
        Node(int key, int val) { this.key = key; this.val = val; }
    }

    private final int capacity;
    private final Map<Integer, Node> map = new HashMap<>();
    private final Node head = new Node(0,0), tail = new Node(0,0);

    LRUCache(int capacity) {
        this.capacity = capacity;
        head.next = tail; tail.prev = head;   // 虚拟节点互联
    }

    public int get(int key) {
        Node node = map.get(key);
        if (node == null) return -1;
        moveToHead(node);    // 访问 → 移到最近端
        return node.val;
    }

    public void put(int key, int value) {
        Node node = map.get(key);
        if (node != null) {
            node.val = value;
            moveToHead(node);
        } else {
            Node n = new Node(key, value);
            map.put(key, n);
            addToHead(n);
            if (map.size() > capacity) {
                Node lru = tail.prev;       // 最久未用
                removeNode(lru);
                map.remove(lru.key);
            }
        }
    }

    private void addToHead(Node node) {
        node.prev = head;  node.next = head.next;
        head.next.prev = node;  head.next = node;
    }

    private void removeNode(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    private void moveToHead(Node node) {
        removeNode(node);  addToHead(node);
    }
}
