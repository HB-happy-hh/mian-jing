## 选择题

### 1. 关于 Unicode 和 UTF-8，下列说法错误的有？

**答案：AC**

- **A 错误**：Unicode 不只是规定符号的二进制代码，它是完整的字符标准，而 UTF-8/16/32 才是具体的存储编码方式。
- **B 正确**：UTF-8 对 ASCII 范围内字符（0~127）完全兼容，字节值一致。
- **C 错误**：UTF-8 是**变长**编码（1~4 字节），每个字符固定 4 字节是 UTF-32 的特征。
- **D 正确**：汉字 Unicode 码点范围在 U+4E00~U+9FFF，UTF-8 编码占 3 字节。

---

### 2. 二进制数 10000001 的反码和补码分别是？

**答案：C（11111110 和 11111111）**

原码 `10000001`（负数，最高位为符号位）：
- **反码**：符号位不变，其余取反 → `11111110`
- **补码**：反码 + 1 → `11111111`

---

### 3. 关于整数和浮点数，说法正确的有？

**答案：AB**

- **A 正确**：整数用二进制补码存储；浮点数按 IEEE 754 分为符号位、指数、尾数三部分。
- **B 正确**：32 位 float 范围约 ±3.4×10³⁸，远大于 32 位 int 的 ±2.1×10⁹。
- **C 错误**：整数存储范围与平台/OS 位数有关（如 C/C++ 中 long 大小因平台而异）。
- **D 错误**：浮点数只能表示有限个离散值，无法表示区间内所有实数。

---

### 4. 队列和栈有什么区别？

**答案：A**

- **队列**：先进先出（FIFO）
- **栈**：后进先出（LIFO）

---

### 5. 平衡二叉树的插入节点操作平均时间复杂度是？

**答案：D（O(log n)）**

平衡二叉树高度为 O(log n)，插入时沿高度查找位置，旋转操作为 O(1)，总计 **O(log n)**。

---

### 6. 现在最快且最通用的排序算法是什么？

**答案：A（快速排序）**

快速排序平均 O(n log n)，常数因子小，缓存友好，是实践中最快的通用排序算法。Java `Arrays.sort`（基本类型）底层用双轴快排。

---

### 7. 设栈的输入序列是 1,2,3,4,5，下列哪个是合法输出序列？

**答案：D（3 2 1 5 4）**

| 操作 | 栈状态 | 输出 |
|---|---|---|
| 压入 1,2,3 | [1,2,3] | — |
| 弹出 3,2,1 | [] | 3,2,1 |
| 压入 4,5 | [4,5] | — |
| 弹出 5,4 | [] | 5,4 |

A/B/C 均在某步骤违反了栈的规则。

---

### 8. 执行一趟快速排序能够得到的序列是？

**答案：A（[41,12,34,45,27] 55 [72,63]）**

原始序列 `55,41,12,34,45,27,72,63`，以首元素 55 为 pivot 执行一趟分区：
- 左侧全部 < 55，右侧全部 > 55 ✓

B、C、D 中右侧或左侧存在不满足条件的元素。

### 9. 先序遍历与层次遍历相同的二叉树是？

**答案：B（单支树）**

单支树（斜树）每层只有一个节点，先序和层次遍历顺序完全一致。

---

### 10. 树高为 6 的平衡二叉树，节点数可能是？

**答案：B（32）**

平衡二叉树高度 h=6 时：
- **最少节点**：N(6) = 20（斐波那契式递推）
- **最多节点**：2⁶−1 = 63

范围 [20, 63] 中只有 32 满足，16<20，64>63，128>63。

---

### 11. 递归计算 fact(n) 需要调用该函数几次？

**答案：n+1 次**

调用链：fact(n) → fact(n-1) → ... → fact(1) → fact(0)，共 n+1 次。

---

### 12. 关于进程和线程的差别，说法正确的有？

**答案：AD**

- **A 正确**：线程是进程内的执行单元，也是调度实体。
- **B 错误**："线程作为分配的基本单位"有误，分配资源的基本单位是**进程**。
- **C 错误**：内核级线程切换也需要经过内核态，并非只在用户态进行。
- **D 正确**：进程间和同一进程内多线程均可并发执行。

### 14. 关于堆 heap 和栈 stack 的差别，正确的有？

**答案：ACD**

- **A 正确**：stack 由 OS 自动管理；heap 由程序员手动 malloc/free。
- **B 错误**：编译期变量分配在数据段或栈，不在堆。
- **C 正确**：stack 默认只有 1~8MB；heap 可使用数 GB 内存。
- **D 正确**：C 语言 `malloc()` 分配的内存在堆上。

---

### 15. 关于浅拷贝 copy 和深拷贝 deepcopy，说法正确的有？

**答案：ABCD（全选）**

- A：浅拷贝只复制引用/指针，不复制对象本身 ✓
- B：深拷贝递归复制对象本身及其所有子对象 ✓
- C：浅拷贝后修改一个会影响另一个（共享内存） ✓
- D：深拷贝后两者完全独立，互不影响 ✓

### 18. C 指针代码输出什么？


### 27. 三国冰河时代（追及问题）

**题意：** 信使在拐角处，斥候距拐角 x，信使需走 y 到终点。斥候速度 v1，信使未被发现时 v2，被发现后 v3。判断信使是否能胜利。

**核心公式推导：**

- 阶段1（斥候未到拐角，时长 x/v1）：信使走 p₀ = v₂x/v₁
- 阶段2（双方在同一直线）：信使到终点时间 (y-p₀)/v₃，斥候追上时间 p₀/(v₁-v₃)
- 信使胜利条件（化简）：**y(v₁-v₃) ≤ v₂x**

```java
import java.util.Scanner;
import java.io.*;

public class Solution {

    public String solve(long v1, long v2, long v3, long x, long y) {
        // 信使胜利条件：y*(v1-v3) <= v2*x
        // v1<=v3 时左边<=0，永远成立（Yes）
        // v1>v3 时需满足上式
        if (y * (v1 - v3) <= v2 * x) {
            return "Yes";
        } else {
            return "No";
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int T = Integer.parseInt(br.readLine().trim());
        StringBuilder sb = new StringBuilder();
        Solution solution = new Solution();
        while (T-- > 0) {
            StringTokenizer st = new StringTokenizer(br.readLine());
            long v1 = Long.parseLong(st.nextToken());
            long v2 = Long.parseLong(st.nextToken());
            long v3 = Long.parseLong(st.nextToken());
            long x  = Long.parseLong(st.nextToken());
            long y  = Long.parseLong(st.nextToken());
            sb.append(solution.solve(v1, v2, v3, x, y)).append('\n');
        }
        System.out.print(sb);
    }
}
```

**样例验证：**
- `4 6 1 10 20`：20×3=60 ≤ 60 → Yes ✓
- `3 100 2 50 100`：100×1=100 ≤ 5000 → Yes ✓
- `10 1 1 100 100`：100×9=900 ≤ 100？No ✓

### 28. 途游笔试 g 函数题

**题意：** 定义 g(x₁,...,xₙ) 为十进制不进位加法（各位数字对应相加后 mod 10）。给定数组 a，构造数组 b，使得对所有前缀 r，g(g(a₁,...,aᵣ), g(b₁,...,bᵣ)) 为质数。

**思路：**

令目标质数为 2。维护前缀 g 和 Aᵣ，构造 Bᵣ 使 g(Aᵣ, Bᵣ) = 2：
- Bᵣ 各位 = (target位 - Aᵣ对应位 + 10) % 10，其中 target = 2（只有个位是2）
- bᵣ = Bᵣ 按位减去 Bᵣ₋₁（mod 10）

由于 aᵢ ≥ 1，bᵣ 必然 ≥ 1；且 bᵣ < 10⁷ ✓

```java
import java.util.*;
import java.io.*;

public class Solution {

    // g(a, b)：按位相加 mod 10，不进位
    static int digitAdd(int a, int b) {
        int res = 0, place = 1;
        for (int i = 0; i < 7; i++) {
            res += ((a / place + b / place) % 10) * place;
            place *= 10;
        }
        return res;
    }

    // 求 B 使 g(A, B) = 2（各位互补）
    static int complement(int A) {
        int[] target = {2, 0, 0, 0, 0, 0, 0}; // 2 的各位
        int res = 0, place = 1;
        for (int i = 0; i < 7; i++) {
            res += (target[i] - (A / place) % 10 + 10) % 10 * place;
            place *= 10;
        }
        return res;
    }

    // 按位相减 mod 10
    static int digitMinus(int a, int b) {
        int res = 0, place = 1;
        for (int i = 0; i < 7; i++) {
            res += ((a / place - b / place + 10) % 10) * place;
            place *= 10;
        }
        return res;
    }

    public int[] solve(int n, int[] a) {
        int[] b = new int[n];
        int A = 0;     // A_r（前缀 g 和）
        int BPrev = 0; // B_{r-1}

        for (int i = 0; i < n; i++) {
            A = digitAdd(A, a[i]);          // 更新 A_r
            int BCurr = complement(A);      // 使 g(A_r, B_r) = 2
            b[i] = digitMinus(BCurr, BPrev); // b_r = B_r 按位减 B_{r-1}
            BPrev = BCurr;
        }
        return b;
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine().trim());
        StringTokenizer st = new StringTokenizer(br.readLine());
        int[] a = new int[n];
        for (int i = 0; i < n; i++) a[i] = Integer.parseInt(st.nextToken());

        Solution sol = new Solution();
        int[] b = sol.solve(n, a);

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(b[i]);
        }
        System.out.println(sb);
    }
}
```

**样例验证（a=[1,1]）：**
- r=1：A=1，B=1，b₁=1，g(1,1)=2 ✓
- r=2：A=2，B=0，b₂=9，g(2,0)=2 ✓（输出 `1 9`，与示例 `1 10` 均为合法答案）
