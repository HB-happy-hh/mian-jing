最大递增子序列和

Maximum Sum Increasing Subsequence · Dynamic Programming

// 核心思路

  • 定义 dp[i] = 以 arr[i] 结尾的最大递增子序列之和
  • 初始化：dp[i] = arr[i]（子序列至少包含自身）
  • 转移：枚举所有 j < i，若 arr[j] < arr[i]，则可以把 arr[i] 接在 j 结尾的子序列后面
  • 答案：max(dp[0..n-1])
dp[i] = arr[i]    （初始值）
dp[i] = max(dp[i],  dp[j] + arr[i])    当 arr[j] < arr[i]，∀ j < i
ans = max(dp[0], dp[1], ..., dp[n-1])
