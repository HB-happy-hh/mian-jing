二叉树中的最大路径和

路径可以从树中任意节点出发，到任意节点结束，不能重复经过同一节点。
核心思路：后序 DFS — 每个节点计算「单侧贡献值」，同时用「穿越路径」刷新全局最大值。

两个核心概念
向上返回 · 只选一侧

① 单侧贡献值

以当前节点为起点，只选左或右一侧向下延伸的最大路径和。
必须单侧，因为父节点还要继续往上拼接，路径不能分叉。
若子树贡献为负，直接取 0（相当于不走那条路）。

gain(node) = node.val + max(gain(left), gain(right), 0)
更新答案 · 不向上返回

② 穿越当前节点的路径

以当前节点为「拱顶」，左弧 + 节点 + 右弧，形成完整路径。
此值仅用于刷新全局最大值。不能向父节点返回，
因为返回了就变成"分叉路径"，不合法。

pathThrough = leftGain + node.val + rightGain → maxSum = max(maxSum, pathThrough)

Java 实现
private int maxSum = Integer.MIN_VALUE;  // 全局最大值，初始化为最小整数

public int maxPathSum(TreeNode root) {
    dfs(root);
    return maxSum;
}

private int dfs(TreeNode node) {
    if (node == null) return 0;

    // ① 递归求左右子树的单侧贡献值，负贡献直接取 0（不走）
    int leftGain  = Math.max(dfs(node.left),  0);
    int rightGain = Math.max(dfs(node.right), 0);

    // ② 穿越当前节点的完整弧线路径，仅用于刷新全局答案，不向上传递
    int pathThrough = leftGain + node.val + rightGain;
    maxSum = Math.max(maxSum, pathThrough);

    // 向父节点只能返回单侧最大贡献（只能选左或右一边）
    return node.val + Math.max(leftGain, rightGain);
}
复杂度分析
时间复杂度
O(n)
每个节点恰好访问一次
空间复杂度
O(h)
递归栈深度 h，最坏退化为 O(n)
