# 3. 无重复字符的最长子串

**难度**：中等

## 题目描述

给定一个字符串 `s`，请你找出其中不含有重复字符的**最长子串**的长度。

## 示例

**示例 1：**
```
输入：s = "abcabcbb"
输出：3
解释：因为无重复字符的最长子串是 "abc"，所以其长度为 3。
```

**示例 2：**
```
输入：s = "bbbbb"
输出：1
解释：因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3：**
```
输入：s = "pwwkew"
输出：3
解释：因为无重复字符的最长子串是 "wke"，所以其长度为 3。
         注意，答案必须是子串，"pwke" 是子序列而不是子串。
```

**示例 4：**
```
输入：s = ""
输出：0
```

## 提示

- `0 <= s.length <= 5 * 10⁴`
- `s` 由英文字母、数字、符号和空格组成

## 思路

使用**滑动窗口 + HashMap** 解决：

- 维护一个窗口 `[left, right]`，窗口内始终保持无重复字符。
- `right` 不断向右扩张，将新字符纳入窗口。
- 当 `right` 指向的字符已在窗口内出现过时，将 `left` 跳跃到该重复字符上次出现位置的下一位。
- 用 `HashMap` 记录每个字符最后出现的索引，实现 `left` 的快速跳跃。
- 每次移动 `right` 后，更新最大长度 `maxLen = max(maxLen, right - left + 1)`。

> ⚠️ `left` 只能前进不能后退，需用 `left = max(left, 上次位置 + 1)` 保证正确性。

**时间复杂度**：O(n)，`left` 和 `right` 各最多移动 n 次  
**空间复杂度**：O(min(m, n))，m 为字符集大小
