括号生成

给定 n 代表括号对数，生成所有可能的合法括号组合。
关键洞察：合法的充要条件 = 任意前缀中左括号数 ≥ 右括号数，且最终左=右=n。

核心剪枝约束
OPEN BRACKET

什么时候可以放 ( ？

已用左括号数 open < n 时，随时可以放。

open < n → 追加 '('
CLOSE BRACKET

什么时候可以放 ) ？

已用右括号数 close < open 时才能放，保证不出现多余的右括号。

close < open → 追加 ')'
TERMINATE

什么时候找到一条合法路径？

当前字符串长度 == 2n（左右各 n 个），直接收入结果。

len == 2n → 加入结果集
CATALAN NUMBER

结果总数是多少？

合法括号串的数量等于第 n 个卡特兰数 C(n)。

C(n) = C(2n,n) / (n+1) n=1→1 n=2→2 n=3→5 n=4→14

方法二：动态规划（卡特兰数递推）

固定第一个字符一定是 (， 设其对应的 ) 在位置 2k+1（1-based）：
 · 括号内部有 k 对 → 取 dp[k] 中的所有组合
 · 右侧剩余 n-1-k 对 → 取 dp[n-1-k] 中的所有组合
拼接方式："(" + inner + ")" + outer

代码实现
GenerateParentheses.java 
public List<String> generateParenthesis(int n) {
    List<String> result = new ArrayList<>();
    backtrack(result, new StringBuilder(), 0, 0, n);
    return result;
}

private void backtrack(List<String> result, StringBuilder current,
                       int open, int close, int n) {
    // 终止：左右括号都放满了
    if (current.length() == 2 * n) {
        result.add(current.toString());
        return;
    }

    // 分支1：放左括号（还没放满）
    if (open < n) {
        current.append('(');
        backtrack(result, current, open + 1, close, n);
        current.deleteCharAt(current.length() - 1); // 撤销
    }

    // 分支2：放右括号（close < open 才合法）
    if (close < open) {
        current.append(')');
        backtrack(result, current, open, close + 1, n);
        current.deleteCharAt(current.length() - 1); // 撤销
    }
}
public List<String> generateParenthesisDP(int n) {
    // dp[i] 存 i 对括号的所有合法组合
    List<List<String>> dp = new ArrayList<>();
    dp.add(List.of(""));   // dp[0] = [""]

    for (int i = 1; i <= n; i++) {
        List<String> cur = new ArrayList<>();
        for (int k = 0; k < i; k++) {
            // "(" + dp[k] + ")" + dp[i-1-k]
            for (String inner : dp.get(k)) {
                for (String outer : dp.get(i - 1 - k)) {
                    cur.add("(" + inner + ")" + outer);
                }
            }
        }
        dp.add(cur);
    }
    return dp.get(n);
}
