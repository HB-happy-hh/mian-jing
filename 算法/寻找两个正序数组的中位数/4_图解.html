寻找两个正序数组的中位数

要求 O(log(m+n))，不能合并排序。核心：在较短数组上二分"切割位置"，
找到使左半 ≤ 右半的合法切割，从切割边界直接算出中位数。

核心思路

中位数的本质

把合并后的有序数组切成等长的左右两半（奇数时左半多一个），中位数就是左半最大值（奇数）或左右边界均值（偶数）。

奇数：median = max(左半最大) 偶数：median = ( max(左半最大) + min(右半最小) ) / 2

切割合法的条件

设 nums1 左半取 i 个，nums2 左半取 j = half−i 个，切割合法当且仅当交叉比较成立：

nums1[i-1] ≤ nums2[j] (左1 ≤ 右2) nums2[j-1] ≤ nums1[i] (左2 ≤ 右1)

二分策略

在较短数组上枚举 i（0 到 m），j 由 i 唯一确定。根据不等式方向调整 lo / hi：

nums1[i-1] > nums2[j] → i 太大 → hi=i-1 nums2[j-1] > nums1[i] → i 太小 → lo=i+1

边界处理

切割位置可以是 0（nums1 全在右半）或 m（nums1 全在左半），越界时用 ±∞ 代替，保证比较逻辑统一。

i=0 → maxLeft1 = −∞ i=m → minRight1 = +∞ j=0 → maxLeft2 = −∞ j=n → minRight2 = +∞

public double findMedianSortedArrays(int[] nums1, int[] nums2) {
    // 始终对较短数组二分
    if (nums1.length > nums2.length)
        return findMedianSortedArrays(nums2, nums1);

    int m = nums1.length, n = nums2.length;
    int half = (m + n + 1) / 2;  // 左半总元素数

    int lo = 0, hi = m;
    while (lo <= hi) {
        int i = lo + (hi - lo) / 2;  // nums1 切割位置
        int j = half - i;              // nums2 切割位置

        int maxLeft1  = (i == 0) ? Integer.MIN_VALUE : nums1[i - 1];
        int minRight1 = (i == m) ? Integer.MAX_VALUE : nums1[i];
        int maxLeft2  = (j == 0) ? Integer.MIN_VALUE : nums2[j - 1];
        int minRight2 = (j == n) ? Integer.MAX_VALUE : nums2[j];

        if (maxLeft1 <= minRight2 && maxLeft2 <= minRight1) {
            // ✓ 合法切割，直接算中位数
            if ((m + n) % 2 == 1)
                return Math.max(maxLeft1, maxLeft2);
            return (Math.max(maxLeft1, maxLeft2)
                  + Math.min(minRight1, minRight2)) / 2.0;
        } else if (maxLeft1 > minRight2) {
            hi = i - 1;  // i 太大
        } else {
            lo = i + 1;  // i 太小
        }
    }
    return 0.0;
}
public double findMedianBrute(int[] nums1, int[] nums2) {
    int m = nums1.length, n = nums2.length;
    int[] merged = new int[m + n];
    int i = 0, j = 0, k = 0;
    while (i < m && j < n)
        merged[k++] = nums1[i] <= nums2[j] ? nums1[i++] : nums2[j++];
    while (i < m) merged[k++] = nums1[i++];
    while (j < n) merged[k++] = nums2[j++];

    int total = m + n;
    return total % 2 == 1
        ? merged[total / 2]
        : (merged[total / 2 - 1] + merged[total / 2]) / 2.0;
}
