# 二分查找 — 返回第一次出现的 target 的索引

## 题目描述

给定一个按**非递减顺序**排列的整数数组 `nums`（可能包含重复元素）和一个目标值 `target`，请你找出 `target` 在数组中**第一次出现**的下标。

如果数组中不存在 `target`，返回 `-1`。

---

## 函数签名

```java
public int searchFirstOccurrence(int[] nums, int target)
```

---

## 示例

**示例 1：**

```
输入：nums = [1, 2, 2, 2, 3, 4, 5], target = 2
输出：1
解释：target = 2 第一次出现在下标 1 处。
```

**示例 2：**

```
输入：nums = [1, 3, 3, 5, 5, 5, 7], target = 5
输出：3
解释：target = 5 第一次出现在下标 3 处。
```

**示例 3：**

```
输入：nums = [1, 2, 3, 4, 5], target = 6
输出：-1
解释：target = 6 不存在于数组中，返回 -1。
```

**示例 4：**

```
输入：nums = [2, 2, 2, 2, 2], target = 2
输出：0
解释：所有元素均为 target，第一次出现在下标 0 处。
```

---

## 约束条件

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 按**非递减顺序**排列
- `-10^4 <= target <= 10^4`

---

## 要求

- 时间复杂度：`O(log n)`
- 空间复杂度：`O(1)`
- 必须使用**二分查找**实现，不得使用线性扫描。
