最大子数组和

给你一个整数数组，找出具有最大和的连续子数组，返回其最大和。
核心思路：Kadane 算法（动态规划的空间优化版本）

核心思路

关键问题：子数组要不要"接上"前面？

遍历到第 i 个元素时，我们定义 curSum = 以 nums[i] 结尾的子数组的最大和。 此时只有两种选择：

dp[i] = max( nums[i], dp[i-1] + nums[i] )

— 如果前面积累的 curSum < 0，带上它只会"拖后腿"，直接从 nums[i] 重新开始。
— 如果 curSum ≥ 0，接上去能让和更大，继续累积。

整个过程只需一个变量 curSum 保存当前状态，另一个 maxSum 记录历史最大值，空间 O(1)。

Java 实现
public int maxSubArray(int[] nums) {
    int curSum = nums[0];   // 以当前元素结尾的子数组最大和
    int maxSum = nums[0];   // 全局最大值

    for (int i = 1; i < nums.length; i++) {
        // 核心决策：接上前面 vs 重新开始
        curSum = Math.max(nums[i], curSum + nums[i]);
        maxSum = Math.max(maxSum, curSum);
    }

    return maxSum;
}
